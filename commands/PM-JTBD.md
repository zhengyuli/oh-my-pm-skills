# PM-JTBD: JTBD 用户研究

Jobs-to-be-Done 深度用户动机分析。

## 调用的技能

```
1. skills/core/jtbd-analyzer
   ↓ 待办任务和四力量分析

2. skills/core/research-assistant
   ↓ 用户研究和验证

3. skills/engines/scoring-engine
   ↓ 动机强度计算

4. skills/engines/template-engine
   ↓ 使用 jtbd-analysis.md 模板输出
```

## JTBD 分析框架

- **主任务识别**: 核心待办任务定义
- **任务拆解**: 子任务和执行步骤
- **四力量分析**:
  - Push (推动力): 现状痛点
  - Pull (吸引力): 解决方案吸引力
  - Anxiety (焦虑感): 采用障碍
  - Habit (习惯性): 现有习惯阻力
- **雇佣标准**: 功能、性能、情感、社会性
- **用户旅程**: 关键时刻 (MOT) 和触点

- 目标用户群体
- 研究方法偏好 (访谈/调查/观察)
