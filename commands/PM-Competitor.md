# PM-Competitor: 竞品分析报告

竞争对手深度分析和战略建议。

## 调用的技能

```
1. skills/core/research-assistant
   ↓ 竞品信息收集和分析

2. skills/engines/scoring-engine
   ↓ 竞争力评分和对比

3. skills/core/priority-evaluator
   ↓ 差异化机会评估

4. skills/engines/template-engine
   ↓ 使用 competitor-analysis.md 模板输出
```

## 分析维度

- **产品分析**: 功能、技术栈、用户体验
- **商业分析**: 定价、收入模式、CAC/LTV
- **客户分析**: 目标用户、市场定位、评价
- **营销分析**: 品牌定位、渠道、增长策略
- **竞争态势**: 市场集中度、进入壁垒

## 开始

请提供：
- 竞争对手列表或产品领域
- 分析深度 (概览/深度/全方位)
