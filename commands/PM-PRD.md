# PM-PRD: 产品需求文档生成

生成完整的产品需求文档 (PRD)。

## 调用的技能

```
1. skills/core/jtbd-analyzer
   ↓ 待办任务和动机分析

2. skills/core/research-assistant
   ↓ 市场和竞品研究

3. skills/core/priority-evaluator
   ↓ 功能优先级 RICE/ICE 评分

4. skills/core/prd-generator
   ↓ 生成 PRD 文档

5. skills/engines/template-engine
   ↓ 使用模板渲染输出
```

## 开始

请提供：
- 产品名称和类型
- 核心功能和目标用户
- PRD 类型选择：lean(精简8节) / standard(标准15节) / security(安全增强17节)
